## Getting started

Just install [uv](https://docs.astral.sh/uv/) and run the following commands in the repo's directory.
```bash
uv sync
uv run <name-of-the-script>
```


## Plotting

```bash
uv run plot.py data/baseline.csv "Baseline" --output docs/img/baseline.svg
```
